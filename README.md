# Setting up locally

## Frontend part
1. Run `npm i` in the "Frontend" folder of this project to install all dependencies
2. Run `npm run start` to start the development server

   
## Backend part
1. Run `dotnet run` in the "API" folder of this project to start the development server

# Settings
URL port for the backend is 53386(https), this port is hardcoded in the frontend too
